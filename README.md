# Code and data for the SIGIR'2022 paper "Item-Provider Co-learning for Sequential Recommendation" (IPSRec)

## Requirements

- Python 3.8
- PyTorch 1.7.0
- RecBole 1.0.0

## Usage

### Run single-stream version IPSRec-S

```sh
sh ipsrecs.sh
```

### Run cross-stream version IPSRec-C

```sh
sh ipsrecc.sh
```
